# Barbarik

Your AI Department. Deployed in 30 Days.

We build and manage teams of AI agents that run your revenue operations 24/7. They source leads, write outbound, create content, recruit candidates, and get smarter every week.

## Live Site

🚀 **https://barbarik.vercel.app**

## Agents

| Agent | Role |
|-------|------|
| Scout | Lead Intelligence |
| Archer | Outbound Sales |
| Pulse | Pipeline Manager |
| Oracle | SEO & Content Intelligence |
| Flash | Content Production |
| Sentinel | Operations & Reporting |

## Tech Stack

- Next.js 14
- Tailwind CSS
- Vercel
