# Barbarik - Autonomous Software Organisation

## Overview

Barbarik is an agentic AI application designed to build autonomous software organizations using LYZR MCP (Model Context Protocol).

## Setup Guide

The complete setup guide is available in `lyzr_mcp_github_setup_guide.pdf`.

### Quick Start

1. Clone the repository
2. Install dependencies
3. Configure LYZR MCP server
4. Run the application

### Documentation

For detailed instructions, please refer to the setup guide PDF.
